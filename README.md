# eslint-config-exmg

> ESLint [shareable config](http://eslint.org/docs/developer-guide/shareable-configs.html)

## Installation

```
$ npm install --save-dev eslint-config-exmg
```

or

```
$ yarn add --dev eslint-config-exmg
```

## Usage

```js
{
  "extends": "exmg",
  "rules": {
    // Optional project specific rules
  }
}
```
